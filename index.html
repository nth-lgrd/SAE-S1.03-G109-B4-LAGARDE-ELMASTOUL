01. Introduction

Ce projet consiste à s'initier à la technique de virtualisation en installant un environnement complet sous Linux.Nous avons dû transformer une machine physique en hyperviseur pour accueillir un serveur Ubuntu Server fonctionnel.

Ce site web est hébergé directement sur la machine virtuelle que j'ai créée, prouvant ainsi la réussite de l'installation du serveur Web Apache.
